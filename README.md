# FanCurier App
## Tema 1

A C++ console application that simulates an e-commerce and courier delivery system. It includes an interactive menu system separated into User and Admin panels, featuring cart management, stock tracking, and courier capacity logic.

## Features

### User Panel
Account Creation: Input basic details (Name, Address, Phone, Email) with regex validation.

Shopping Cart: Browse available products, add items to the box, and remove them if you change your mind.

Order Placement: View total price/weight and select an available courier.

Dynamic Couriers: Couriers have varying max weight capacities and tier levels (VIP orders require higher-tier couriers).

### Admin panel
Inventory Management: Add new products, delete existing ones, or edit current stock variables (Name, Stock, Weight, Price).
